## How to use

We're using Love2D 11.5, coded in plain Lua with [LuaLS](https://github.com/LuaLS/lua-language-server) annotations for type checking.

To run: `make run`

## How to typecheck

Install the [lua-language-server](https://github.com/LuaLS/lua-language-server) and run `make typecheck`. Using the VSCode Lua extension should just work, too.

## How to test

Run `make test`
